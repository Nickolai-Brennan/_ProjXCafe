# ☕ ProjXCafe — Brand Style Guide (v1)

Built around your **coffee cup + nerd glasses logo** and aligned with your product direction (social + proof-of-work platform) 

---

# 1. Brand Foundation

## Brand Identity

ProjXCafe =
**Smart builders solving real problems in a social workspace**

## Personality

* Intelligent but approachable
* Fast, not corporate
* Technical but not intimidating
* Social, not transactional

## Tone

* Direct
* Builder-focused
* Slightly playful (not childish)

---

# 2. Logo System

## Primary Logo

* Coffee cup + nerd glasses + subtle glow
* Used on:

  * landing page
  * marketing
  * auth screens

## Icon Logo

* Cup + glasses only
* No text
* Used for:

  * navbar
  * favicon
  * mobile app

## Wordmark

* “ProjXCafe” text only
* Used when space is limited

---

## Logo Rules

### Do

* Use on dark backgrounds (preferred)
* Keep glow subtle
* Maintain clean outline

### Don’t

* Don’t stretch or distort
* Don’t add extra colors
* Don’t place on busy backgrounds
* Don’t over-glow (kills readability)

---

# 3. Color System

## Primary Palette (Core Identity)

* Background: `#0B0F14`
* Surface/Card: `#121821`
* Border: `#1F2937`

---

## Accent Colors (Brand Energy)

* Cyan (Primary): `#00E5FF`
* Purple (Secondary): `#7C3AED`
* Orange (Highlight): `#FF7A00`

Use sparingly for:

* buttons
* highlights
* badges
* hover states

---

## Neutral Text

* Primary Text: `#E6EDF3`
* Secondary Text: `#9CA3AF`
* Muted: `#6B7280`

---

## Status Colors

* Success: `#22C55E`
* Warning: `#F59E0B`
* Error: `#EF4444`

---

# 4. Typography

## Headings

**Barlow Condensed**

* Strong
* Compact
* Works well with tech UI

## Body

**Inter**

* Clean
* Highly readable

## Code / Tech Labels

**JetBrains Mono**

* Used for:

  * skill tags
  * repo names
  * technical metadata

---

# 5. UI Design Language

## Core Principles

* Card-based layout
* Dense but readable
* Minimal borders
* Soft glow accents
* Grid + feed hybrid (Reddit + marketplace)

---

## Components

### Cards

* Background: `#121821`
* Border: subtle (`#1F2937`)
* Hover: slight lift + glow

---

### Buttons

Primary:

* Background: cyan gradient
* Text: dark
* Slight glow on hover

Secondary:

* Outline
* Light text
* Subtle hover fill

---

### Tags / Pills

* Rounded
* Use color coding:

  * frontend = cyan
  * backend = purple
  * design = orange

---

### XP / Badge Elements

* Small glow rings
* Icon + label
* Gradient fill optional

---

# 6. Iconography

## Style

* Outline icons
* Rounded edges
* Minimal detail
* Consistent stroke width

## Icon Categories

* Skills (frontend, backend, db)
* Actions (submit, approve, claim)
* Social (GitHub, LinkedIn, etc.)
* Status (verified, trending, xp)

---

# 7. Imagery Style

## Visual Direction

* Dark UI screenshots
* Neon highlights
* Real product usage
* Minimal stock photos

## Avoid

* generic corporate imagery
* overly abstract art
* cluttered visuals

---

# 8. Motion & Interaction

## Microinteractions

* Button glow on hover
* Card lift on hover
* XP gain animation (quick burst)
* Submission success pulse

## Timing

* Fast (150–250ms)
* No heavy animations
